# Segundo-Parcial

Este programa es un sistema sencillo de inventario hecho en C# donde el usuario primero debe elegir qué tipo de producto quiere registrar: producto electrónico o producto de alimento, después de elegir la opción se le pide que introduzca los datos del producto como nombre, código, precio y la cantidad. 
y dependiendo de los productos electronicos tienen garentia y los productos de alimentos tienen fecha de caducidad.
